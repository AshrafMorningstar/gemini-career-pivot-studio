## Description

The Gemini Career Pivot Studio is an innovative web application designed to empower individuals undergoing career transitions. Leveraging the advanced capabilities of the Google Gemini API, this studio provides intelligent assistance, personalized insights, and actionable guidance to help users navigate their professional journey effectively and confidently. Whether you're exploring new industries, upskilling for a promotion, or entirely reinventing your career path, this tool aims to be your trusted AI-powered catalyst.

## Features

*   **Personalized Career Guidance**: Receive tailored recommendations based on your skills, experience, and aspirations.
*   **Skill Gap Analysis**: Identify areas for development and suggest relevant learning resources.

## Installation

To get this project up and running on your local machine, follow these steps:

### Prerequisites

Make sure you have Node.js installed on your system.
*   [Node.js](https://nodejs.org/) (LTS version recommended)

### Steps

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/gemini-career-pivot-studio.git
    cd gemini-career-pivot-studio
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Set up environment variables**:
    Create a `.env.local` file in the root of the project and add your Google Gemini API key:
    ```
    GEMINI_API_KEY=your_gemini_api_key_here
    ```
    You can obtain a Gemini API key from [Google AI Studio](https://ai.google.dev/gemini).

## Usage

After completing the installation, you can run the application locally:

1.  **Start the development server**:
    ```bash
    npm run dev
    ```
